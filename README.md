### 分支

#### main

- 【版本】：初始範例
  - 由 `CodeSandbox` 的 `JSON-SERVER-STARTER`　修改而來

- 原版網頁移動至 `<public>/<example>` 底下
  - 原版資料集：`<data>/sample.json`

---

#### feature_json-server

- 【版本】：顯示自訂資料到前台畫面上

- 資料：`<data>/db.json`

- 路由：`/api/posts`

---

#### feature_json-server-auth

- 【版本】：加入 auth

- 資料：`users`

- 註冊路由：`/signup`

- 登入路由：`/login`
